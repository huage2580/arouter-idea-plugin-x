# arouter-idea-plugin-x
support kotlin &amp; Android studio4.1;  
support usages;  
multiple target select dialog;  

支持kotlin；  
同个路径的多实现弹窗选择；  
在Route注解点击查看使用列表；  
支持常量定义的路径跳转；  

阿里的朋友一直不更新插件，只能自己更新了，请到右边Release下载
